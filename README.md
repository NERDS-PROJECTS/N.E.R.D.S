# N.E.R.D.S. 🤖

**Robotics Club of NIT Silchar**

Welcome to the official repository of N.E.R.D.S. (NITS Embedded and Robotics Development Society), the premier robotics club of National Institute of Technology, Silchar.

## 🚀 About Us

N.E.R.D.S. is a community of passionate engineers, developers, and innovators dedicated to advancing robotics and automation technology. We foster a collaborative environment where students explore cutting-edge technologies, participate in competitions, and work on real-world projects.

### Our Mission
- 🔬 Promote research and development in robotics and AI
- 🏆 Participate in national and international robotics competitions
- 🎓 Provide hands-on learning experiences to students
- 🌟 Bridge the gap between theoretical knowledge and practical implementation

## 🏃‍♂️ Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn
- Git

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/swarupecenits/N.E.R.D.S.git
   cd N.E.R.D.S
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Set up environment variables**
   ```bash
   cp .env.example .env
   # Configure your environment variables
   ```

4. **Start the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

5. **Open your browser**
   Visit `http://localhost:5173` to see the application.
